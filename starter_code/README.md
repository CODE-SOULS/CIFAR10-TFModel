# Run main script (Instructions)

## For training 

```python
  python main.py --mode "train" --data_dir "./data" --save_dir "./" 
```

## For testing

```python
  python main.py --mode "test" --data_dir "./data" --save_dir "./" 
```

## For predictions

```python
  python main.py --mode "predict" --data_dir "./data" --save_dir "./" 
```
